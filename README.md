# NGWORKS心理危机干预系统
NGWORKS Psychological Crisis Intervention System

## 什么是NGWORKS心理危机干预系统
What is NGWORKS Psychological Crisis Intervention System
本项目主要基于哔哩哔哩社区功能，为心理疾病患者提供了一套检测，识别，收集，预警，处理的办法。
也就是说，这个项目分为四大模块：
1. 对所有评论区数据的收集
2. 识别“树洞”评论区
3. 对需要监控的对象进行监控
4. 响应
